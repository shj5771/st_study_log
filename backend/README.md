# Backend 학습 로그

##  학습 목표
- FastAPI를 활용한 현대적인 백엔드 개발 역량 습득
- RESTful API 설계 및 구현 능력 향상
- 데이터베이스 설계 및 ORM 활용 마스터

##  기술 스택
- **Framework**: FastAPI
- **Database**: PostgreSQL, SQLite
- **ORM**: SQLAlchemy
- **Authentication**: JWT, OAuth2
- **Testing**: pytest
- **Documentation**: Swagger/OpenAPI
